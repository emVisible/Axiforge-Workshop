# Changelog

## [0.2.0] - 2026-06-22

### 新增
- 六层角色模型：轮廓、举止、内心、锚点、轨迹、羁绊
- 折叠面板编辑器，支持逐层填写和进度显示
- 标签系统：独立标签表、自动颜色生成、标签云筛选
- 角色搜索：防抖搜索 + 标签组合筛选 + 排序
- Prompt 导出：System Prompt / 纯文本 / Markdown 三种格式
- 关系系统：预设关系（亲属/情感/社会/创作）+ 自定义关系
- D3 力导向关系图谱，软木板风格，支持拖拽和缩放
- 角色搜索选择器，添加关系时搜索选择
- 草稿自动保存：800ms 防抖存储，刷新页面可恢复
- 骨架屏加载状态

### 变更
- 数据模型从 core/layers/dynamics 重构为六层结构
- 名称字段统一到 contour.name，不再独立存储
- 标签从字符串数组改为独立关联表
- 详情页重新设计信息层级，四 tab 切换
- 视觉统一：圆角 2xl、统一间距、过渡动效、通用 UI 组件库

### 移除
- 旧的情感触发器字段
- 独立的 name 字段（合并到 contour）
